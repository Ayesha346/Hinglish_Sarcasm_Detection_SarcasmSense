# SarcasmSense: A Hinglish Sarcasm Detection Model for Indian Social Media
SarcasmSense focuses on detecting sarcasm in Hinglish social media text using both traditional machine learning and deep learning approaches. A balanced dataset of 3,000 code-mixed sentences was collected and preprocessed to handle noise and linguistic variability. Traditional models such as Logistic Regression and Random Forest were implemented as baseline approaches, followed by fine-tuning a multilingual BERT model for improved performance. The system includes data preprocessing, tokenization, feature engineering, model training, and evaluation using metrics such as accuracy, precision, recall, F1-score, and confusion matrix. The results demonstrate that transformer-based models are effective for handling sarcasm detection in low-resource, code-mixed language settings.
